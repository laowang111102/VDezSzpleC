# 学生宿舍管理系统 java318

## 项目概述

学生宿舍管理系统是一款基于Springboot和Vue.js开发的，旨在提高宿舍管理的效率和便捷性。该系统实现了前后端分离，具有界面美观、操作简便的特点。

## 技术栈

- 后端：Springboot
- 前端：Vue.js

## 功能模块

- 宿舍楼管理
- 房间管理
- 学生信息管理
- 缴费管理
- 报修管理
- 数据统计与分析

## 系统角色

- 管理员：负责整个宿舍楼的管理工作
- 学生：查询个人信息、宿舍费用、报修申请等

## 运行环境

- 服务器：Java 1.8 或更高版本
- 数据库：MySQL 5.7 或更高版本
- 客户端：现代浏览器

## 部署步骤

1. 准备运行环境（Java、MySQL等）
2. 导入数据库文件
3. 部署后端服务（Springboot应用）
4. 部署前端应用（Vue.js项目）

## 目录结构

```
项目根目录
├── backend                # 后端代码（Springboot项目）
├── frontend               # 前端代码（Vue.js项目）
```

## 核心亮点

- 前后端分离，易于维护和扩展
- 提供完善的宿舍管理功能，提高管理效率
- 数据统计与分析，为决策提供依据
